## :round_pushpin: About The Project

&ensp;_This is a braille shape generator web service which generate shapes, numbers, and letters as for the requirements the users put in. A tool that can draw various shapes for their newly constructed 3D printer that allows blind people to touch and feel drawings. SOAP method was chosen as the architectural type for the creation of this system. The service allows the user to select shapes, numbers, and letters from a dropdown list and so it can be converted into braille._<br/>
